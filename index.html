Login

Login to neonion using a predefined test account or sign up as a new user.

Import a document to annotate

Within your workspace upload a textfile *.txt or PDF you want to annotate.

User Interface

tool bar

Provides several tools to support the annotation process.

  • Home: brings you back to the start page
  • NER: automatically assigns concepts to specific text parts
  • Contributers: display or hide the annotations of contributers
  • Annotation Mode: choose between Commenting, Highlighting and Concept Tagging
  • Jump to last Annotation: brings you to the last created annotation in this text

context bar

Shows detailed information about this specific document.

  • number of annotations
  • number of contributers
  • title
  • visibility (private, public or group)
  • user - shows the user ID of the person logged in

Start Annotating

Annotation Modes

  • Commenting (orange): adds a comment to the selected text. Shortcut: ﻿CTRL+ALT+A
  • Highlighting (yellow): highlights the selected text. Shortcut: ﻿CTRL+ALT+S
  • Concept Tagging (green): assigns a concept to the selected text. Shortcut: ﻿CTRL+ALT+D

Concept Tagging

Text parts assigned to the concept person can be further identified with a unique entity (such as a specific person) by using a linked data identifier such as the Wikidata Identifier. If there’s no matching suggestion, the person can be categorized as Unknown Resource.

Here, we link Otto Hahn to Otto Hahn *1879-03-08 deutscher Chemiker und Nobelpreisträger from Wikidata.

In the third step, we can create a link between two entities through a predicate. The possibile predicates for each concept were previously defined in a vocabulary.

group concept

Working together on a document is enabled through the concept of groups. Click on Users in the navigation menu and select Groups. On this page, you have the possibility to create a new group or choose an existing one to add users to it.

Vocabularies

You can choose an existing vocabulary or create your own before you start annotating. Every vocabulary must at least contain one concept and every concept can have assigned properties.

Create a new property

Change to the tab Properties and click on the button Create Property.

Fill in a label, a description and assign concepts that should have this property. You can additionally add the inverse property.

In the following example, the property is member and the assigned concept Person. If we construct an inverse property called member of with the assigned concept Institution, we are then able to construct two relationships:

  1. Person is member of an Institution
  2. Institution has member Person

Create a new concept

Change to the tab Concepts and click on the button Create Concept.

Create your own vocabulary

Click on Vocabulary in the navigation menu to see the currently existing vocabularies. You can create a new one if required.

Fill in a label, a description and add at least one concept to your new vocabulary.

In the following example, we create the vocabulary research institutions. This vocabulary contains two concepts: Person and Institution.
